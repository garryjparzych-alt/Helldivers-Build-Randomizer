# Helldivers-Build-Randomizer
A Build Randomizer for Helldivers 2 in a single HTML file.  If you ever find yourself having a hard time trying different loadouts, this is for you!  You can choose Random, Balanced, and several themed builds (Such as Chaff Clearing, Defense, Anti-Tank), etc.  It will need some more work, but please let me know what you think!
